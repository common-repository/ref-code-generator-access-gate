Generate unique code for users to allow them to access your website.

== Description ==

This plugin generates unique codes and creating custom field in user profile called `Ref code`.
You can enable access gate, to allow users to access website only with their unique ref code.
Access gate is just post with shortcode inside, so you can create whatever text/design you want.

== Installation ==

1. Upload `ref-code-generator` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Ref Code Generator' Options and change your plugin settings.
4. Go to 'Ref Code Generator' and generate your unique reference codes.

== Frequently Asked Questions ==

`Does it protect my website from all access?`

Users cannot access the website content if access gate is enabled. Only header and footer is visible, content part is hidden.

`How can i use it?`

You can create unique code, give it to your users to allow them to access your content. You can also create some discount functions for users.
